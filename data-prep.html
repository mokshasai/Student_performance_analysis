Data Preparation & Exploratory Data Analysis (EDA)

This section describes how the data for this project was gathered, cleaned, and explored. The goal of this stage is to understand the structure, quality, and key characteristics of the data before applying any analytical or modeling techniques.

Data Source

The primary dataset used in this project is the Student Performance Dataset, obtained from the UCI Machine Learning Repository. The dataset contains academic, demographic, social, and behavioral information for secondary school students enrolled in a mathematics course. Each row represents an individual student, and each column represents a specific attribute related to performance or background.

Dataset source: UCI Machine Learning Repository
File used: student-mat.csv

Link to Raw Dataset (UCI Repository)

API Data Collection (Contextual Data)

In addition to the primary dataset, external contextual data was collected using the World Bank Open Data API. This API was used to retrieve country-level education and technology indicators in order to provide broader educational context. These indicators were used for contextual understanding and were not merged at the individual student level.

API website: World Bank Open Data API
Example API endpoint: 

        https://api.worldbank.org/v2/country/PRT/indicator/IT.NET.USER.ZS?format=json

The API data was accessed using Python and processed from JSON format.

Link to World Bank API Documentation

Data Cleaning and Preparation

Data cleaning and preparation were performed using Python to ensure the dataset was accurate, consistent, and suitable for exploration. The dataset was inspected for missing values, incorrect entries, and duplicate records. No missing values were found in the dataset, and all data types were verified for correctness.

Additional preparation steps included reviewing numerical ranges to identify potential outliers and creating a derived performance category based on the final grade. These steps ensured that the data was well-structured and ready for exploratory analysis.
